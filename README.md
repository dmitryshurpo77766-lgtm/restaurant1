# 🍝 FoodRest — Landing Page ресторана

Современный адаптивный landing page для ресторана итальянской кухни.  
Проект выполнен на чистом HTML и CSS без использования фреймворков.

---

# 🚀 Возможности проекта

- Адаптивный дизайн
- Современный UI/UX
- Hero-блок с изображением блюда
- Навигационное меню
- Карточки популярных блюд
- Google Maps интеграция
- Красивый footer
- Hover-анимации
- Flexbox layout
- Полностью responsive интерфейс

---
